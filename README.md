# Ticket Tool
Tool to store the ticket details, view and export ticket data; fields easily configurable within the application

### Start application
> For Unix system
```
sh start_app.sh
```
> For Windows
```
win_start.bat
```

### URL
```
http://localhost:9191/ticket-tool
```

For tool insights
```
http://localhost:9191/ticket-tool/v/insights
```
